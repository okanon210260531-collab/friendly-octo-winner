# Rixon株式会社 Webサイト 最終調整版

この版では、文字のかぶり・かぶりかけを防ぐために、全ページの余白、文字サイズ、折り返し、スマホ表示を再調整しています。

## 修正内容
- ヘッダーのメニューがロゴや他の文字と重ならないよう調整
- アプリ開発ページの「iOSアプリ制作」「Androidアプリ制作」周辺の表示を調整
- 見出し・カード内テキスト・ボタンの文字かぶりを防止
- かぶりかけになりやすい箇所は文字サイズと余白を調整
- スマホ幅でも不自然に詰まらないよう、カードを1列表示へ調整
- メールアドレスや電話番号などの長い文字列がはみ出さないよう調整

## GitHub Pagesへの置き方
1. ZIPを解凍します。
2. 中身をすべてGitHubのリポジトリ直下にアップロードします。
3. `index.html` がリポジトリ直下にある状態にしてください。


## 追加修正
- 見出しを意味のまとまりごとに折り返すよう調整しました。
- 文字がかぶりそうな箇所を再調整しました。
- アプリ開発ページを含む全HTMLを再チェックしました。
